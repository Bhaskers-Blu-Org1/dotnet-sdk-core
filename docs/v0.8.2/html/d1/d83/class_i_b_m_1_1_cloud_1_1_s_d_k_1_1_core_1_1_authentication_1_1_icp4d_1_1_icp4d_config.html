Detailed Description

Options for authenticating via the ICP4D (IBM Cloud Private for Data) token service. This authentication mechanism allows an ICP4D access token to be obtained using a user-supplied username, password and URL. The ICP4D access token is then specified as a Bearer Token within outgoing REST API requests.
